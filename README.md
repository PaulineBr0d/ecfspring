### Projet Backend Spring
Le projet consiste à concevoir la couche Data et la couche contrôleur d'une application de mise en relation (détail des fonctionnalités ci dessous) en utilisant Spring Boot et JPA.

#### L'Application
Application de mise en relation entre des porteurs et porteuses de projet et des dév. Des personnes souhaitent qu'on leur crée une application, elles viennent sur la plateforme pour décrire leur projet et leurs contraintes. Des dév' peuvent voir les projets en attente et candidater pour les projets en questions. La personne ayant mis le projet en ligne viendra ensuite valider la ou les candidatures et lancer le projet une fois que tout le monde est d'accord.

Fonctionnalités cf. uses cases

#### Réalisation attendue
Diagramme d'entité de la base de données, plutôt en UML (via un diagramme de classe), avec les relations entre les entités et leurs propriétés
- Création d'un projet Spring Boot Web JPA
- Création des entités annotées correctement
- Création des repositories JPA avec les méthodes spécifique
- Création de la couche contrôleur pour les différentes fonctionnalités avec les DTO
